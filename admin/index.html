ADMIN LOGIN · 登录

用 GitHub 令牌登录

道序后台用 GitHub 仓库直接存内容。你需要一个 fine-grained Personal Access Token（PAT），只给本仓库的 contents 读写权限。

第一次使用？三步拿到 PAT：
1. 打开 GitHub PAT 设置
2. Repository access 选 764496864/daoxu-site，Permissions → Repository → Contents 设为 Read and write
3. 生成后的 token 粘贴到下面（github_pat_... 开头）
存储在你浏览器本地（localStorage），只有你自己能用。不会上传到任何地方。

内 容 管 理

选择一个文件开始编辑
Markdown 格式： **加粗** · *金色强调* · [链接文字](/xx.html) · `代码样式` · 回车即换行
不懂也没关系——直接当普通文字写，最多就是不会加粗加色。
改好后点"保存"，会自动 commit 到 GitHub，约 60 秒后 daoxu.com.cn 生效。
